# 블로그 첫 게시글이다.

개인적인 이야기를 꺼내는 것에 익숙하지 않아 짧은 단문이 될 것 같습니다..

작년 2023년 7월 바로고를 끝으로 잠시 개발일을 떠나있었습니다.  
집안 사정을 이유로 진로를 바꿔야겠다는 생각을 하게되면서 이렇게 되었는데 생각보다 그 일이 많이 맞지 않았습니다.  
개발일을 할 때 느꼈던 성취감이나 의욕이 잘 생기지 않았던 것이 가장 큰 이유였던 것 같습니다.  
결국 다시금 이 길에 들어가고자 차근차근 준비를 시작하면서 이 블로그를 만들게 되었습니다.

새롭게 준비를 시작하면서 요즘 가장 인기있는 프레임워크인 [NextJS](https://nextjs.org/)도 익히면서  
최근 버전업이 많이된 [React](https://ko.legacy.reactjs.org/)도 이전에 하던 방식이 아닌
최신 React 프로그래밍을 익히고 싶었습니다.

물론 두 가지 프레임워크 / 라이브러리 외에도 Tailwind CSS같은 인기 라이브러리도 사용해보게 되었는데,  
결론적으로 이런 스택들을 적용할 플레이 그라운드로 가볍게 시작할 만한 것이 블로그가 최적이라고 생각했습니다.

이 블로그가 다시한번 개발 열정을 불타오르게해줄 용광로가 되었으면 하면서 첫번째 글은 마치려고 합니다...
